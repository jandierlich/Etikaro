Info

Über Etikaro

Etikaro liest Etiketten von Flaschen, Gläsern und Schachteln per Texterkennung (OCR) direkt auf diesem Gerät aus und legt daraus einen durchsuchbaren Bestand an.

Datenschutz

Alle Fotos, Texte und Daten verbleiben ausschließlich lokal auf diesem Gerät (IndexedDB). Es findet keine Übertragung an einen Server statt. Die Texterkennung läuft vollständig im Browser.

Verwendete Software

Texterkennung: Tesseract.js (Apache-2.0), Sprachdaten Deutsch (Apache-2.0). PDF-Erstellung: jsPDF (MIT-Lizenz). Alle Bestandteile sind Teil dieses Projekts und werden lokal ausgeliefert — keine Drittanbieter-Verbindungen. Diese App ist quelloffen und darf frei an andere weitergegeben werden.
